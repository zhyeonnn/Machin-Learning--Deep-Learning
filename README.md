#### 지도 학습
- 가장 많이 사용되는 것은 합성곱 신경망이다.
- 합성곱 신경망을 목적에 따라 이미지 분류, 이미지 인식, 이미지 분할로 분류한다.

|이미지 분류|이미지 인식|이미지 분할|
|:---:|:---:|:---:|
